## Description

* Pa8 API build on [Nest framework](https://github.com/nestjs/nest)  TypeScript starter repository.
* Swagger OpenAPI
* Jenkins Updating
* JWT Authentication ( updating )

## How to use
Need Docker installed
```bash
$ docker-compose up --build
```

## API Documentation

```bash
$ http://localhost:3000/api
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# incremental rebuild (webpack)
$ npm run webpack
$ npm run start:hmr

# production mode
$ npm run start:prod
```
